# Quick Start

## 1) Install
- Run `setup.bat` (recommended), or
- Create a venv and run `pip install -r requirements.txt`

## 2) Start Backend
```
cd backend
python app.py
```

## 3) Open Frontend
Open `frontend/index.html` in your browser.

## 4) Basic Flow
1. Upload a PDF in Upload tab.
2. Generate MCQs in MCQ tab.
3. Review analytics and run assessments.

## 5) Switch Mode (Optional)
Use the API to switch between `online` and `offline` mode.
See API_EXAMPLES.md.
